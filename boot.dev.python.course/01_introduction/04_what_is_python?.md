## What is Python?

Hundreds of thousands of students start their coding journey right here
with (Python)[https://www.python.org/].

"""
[Lane] Python doesn't have to be your first language, but _I_ think it's
the best first language for most developers. It gets out of your way so
you can focus on learning fundamental programming concepts.
"""

But just because Python is simple doesn't mean it's not useful! Python
is an (Extremely
Popular)[https://survey.stackoverflow.co/2025/technology#1-programming-scripting-and-markup-languages]
language in the industry, and is well-known for:

- Backend web servers
- DevOps and cloud engineering
- Machine learning
- Scripting and automation
- etc...

On the other hand, it _isn't_ known for __frontend development__ or
graphical user interfaces. It's _possible_, but not common.

```question
What's Python not really used for?
- Machine learning
- Frontend apps
- Backend servers
- Automation scripts
```

## Proper Answer:
```
Frontend Apps
```
